# Assets Folder

Place your logo files here:

- **Logo (SVG recommended)**: `public/assets/images/logo.svg`
- **Logo (PNG/JPG)**: `public/assets/images/logo.png` or `logo.jpg`
- **Favicon**: `public/favicon.ico`

## Supported Formats

- SVG (recommended - scalable, small file size)
- PNG (with transparency)
- JPG/JPEG

## Usage

The logo will automatically appear in:
- Homepage header
- Admin dashboard header
- Customer dashboard header
- Login/Register pages

## File Naming

Use one of these names:
- `logo.svg` (recommended)
- `logo.png`
- `logo.jpg`

The app will automatically detect and use the logo.
